Bienvenido al OVI vinculación de una base de datos con una página PHP


En el mundo de las páginas y aplicaciones web el uso de datos es muy importante y necesario, por eso como profesionales en formación, debemos tener la capacidad de conectar nuestras aplicaciones a una base de datos y poder guardar la información tanto de la aplicación, como de nuestros usuarios.

En este OVI mostraremos como lograr que una aplicación o página web con PHP, se conecte a una base de datos, en este caso MYSQL MariaDB, y gestionar la creación, lectura, actualización y eliminación de datos (CRUD), desde nuestros formularios web.

Para lograra este objetivo debemos contar con un servidor apache XAMP o WAMP instalado, el cual incluye el intérprete de código para PHP, una instancia de base de datos MariaDB, y un editor de código de tu preferencia.

Recuerda seguir paso a paso en su orden de aparición, cada uno de los temas que encuentras en el panes de navegación a tu izquierda. Te recomendamos echar una ojeada a las pestañas que encuentras en esta página con los títulos de PHP y MYSQL MariaDB, para que tengas una idea clara de estos dos importantes componentes para nuestro aprendizaje.

Éxito.



PHP

PHP es un lenguaje de programación de propósito general del lado del servidor, el cual es aplicado en el desarrollo web de contenido dinámico, aportando mucho a la evolución del concepto de aplicaciones en línea.
Su aplicación es muy sencilla y se puede incrustar en paginas web, coexistiendo con las etiquetas propias del lenguaje de marcado HTML, y su facilidad de implementación y escritura lo han posicionado como uno de los lenguajes fuertes de páginas y aplicaciones web.

Al momento de realizar esta OVI, la versión de PHP es la 7.3.9. Para más información haz clic en los siguientes imágenes.




MySQL MariaDB


Es un sistema de gestión de bases de datos derivado de MySQL, de licencia GPL, desarrollado y sostenido por el fundador de MySQL Michael (Monty) Widenius.
Actualmente es uno de los motores de base de datos de código abierto y gratuito más populares del mundo, y su instalación viene incluida en el paquete de instalación del servidor apache XAMP, con el cual vamos a realizar los ejemplos que encontraras en este OVI.
