Objetivo



El objetivo de la presente sección es aprender a realizar un archivo de configuración de conexiones a bases de datos con PHP, a fin de poder usar esas propiedades y métodos de conexión para realizar las operaciones CRUD en el motor de base de datos a través de nuestro formulario web.
Empecemos…


Para esta sección deberás usar el editor de código que mas te guste, en este ejercicio utilizamos Visual studio Code, el cual es sumamente ligero, fácil de utilizar y es multiplataforma y multi lenguaje.
Para comenzar ve a la ruta donde instalaste tu servidor Xampp, y dentro del directorio htdocs, crea un nuevo directorio nombrándolo MyCRUD.
Una vez creado el directorio en tu editor de código creas un archivo de extensión .php dentro de este directorio, llamado conexión, es decir conexión.php.
Desde Visual Studio Code simplemente haces clic derecho en el explorador de Windows y haces clic en Open with Code.



y con el editor abierto simplemente haces clic en nuevo y pones el nombre de tu archivo y su extensión, desde el panel izquierdo sobre el nodo nombrado con el nombre del directorio que acabamos de crear, recuerda esta es nuestra carpeta de proyecto.




Al crear tu archivo se abrirá en la parte derecha, en el área de edición de código, y simplemente pegas el siguiente código. 



            <?php
             /*Datos de conexion a la base de datos*/
             $db_host ="localhost";// Direccion IP donde nos conectaremos
             $db_user ="fatobonq";// Usuario de base de datos que creamos
             $db_pass ="#M4g4ly2108$";//contraseña que creamos
             $db_name= "cursos";//nombre de la base de datos que creamos
              
            //La variable $con  ensambla los argumentos que usa la función mysli_connect
            $con = mysqli_connect($db_host, $db_user, $db_pass, $db_name);
              
             if(mysqli_connect_errno ()){
              echo 'No se pudo conectar a la base de datos : '.mysqli_connect_error();
              }
              ?>




Guarda el archivo y ya está, podemos continuar.
