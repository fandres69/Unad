Que aprenderemos



En esta sección aprenderemos a crear una base de datos, una tabla en la base de datos, y un usuario de base de datos con el cual nos conectaremos desde nuestro formulario WEB.
Empecemos…



Ingresamos a nuestro localhost (localhost:80), después ingresamos a phpMyAdmin vamos a crear un usuario de SQL haciendo clic en cuentas de usuario.


Después hacemos clic en agregar cuenta de usuario en la sección nuevo.



Llenamos los datos que nos son solicitados (Usuario, contraseña), en la parte de nombre de Host lo dejamos default, en la sección de crear base de datos para la cuenta de usuario, dejamos las casillas sin activar, y activamos los privilegios solo sobre datos y estructura; el resto de las opciones las dejamos así. Cuando terminamos hacemos clic en el botón continuar de la parte inferior derecha, y eso es todo, ya tenemos un usuario de base de datos.


Si haces clic sobre la imagen ella se amplia



Ahora en el panel del lado izquierdo creamos una base de datos haciendo clic en nueva.



Le damos nombre a nuestra base en y le establecemos el idioma, y hacemos clic en crear.



Una vez creada nuestro panel de bases de datos (panel izquierdo), se actualiza y podemos ver nuestra base en el árbol de esquemas, y en la pantalla central se abrirá la opción de crear tabla, opción con la cual crearemos una tabla denominada cursos_ofertados, con tres (3) columnas, denominadas NombreCurso, ValorCurso, y IdCurso.



Recuerda hacer clic en el botón continuar para crear la tabla.







Al hacer clic en continuar, veremos los campos a llenar para crear nuestras columnas, las cuales llenaremos como lo muestra la siguiente imagen.



Si haces clic en previsualizar SQL podrás ver el script de creación de la tabla, y para guardar la tabla simplemente haces clic en guardar.



Cuando hallamos grabado nuestra tabla pasaremos a insertar unos datos de prueba, hacemos clic en SQL, luego hacemos clic en el botón insert, y remplazamos la consulta que está en pantalla, por el siguiente script, haciendo clic en continuar una vez hallamos remplazado la consulta.


INSERT INTO `cursos_ofertados` (`NombreCurso`, `ValorCurso`)
VALUES
('Excel Basico',65000),
('Excel Intermedio',90000),
('Excel avanzado',120000)





En hora buena ya hemos preparado nuestra base de datos para realizar un CRUD a través de un formulario WEB.
