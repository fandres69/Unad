Bienvenido al OVI Vinculación de una BD a una pagina con PHP



Estimado amigo o amiga, tenga una cordial bienvenida a este OVI, esperando que sea útil y que le sirva ya sea en su etapa de aprendizaje y preparación, o para reforzar un conocimiento previo del tema aquí abordado presentamos el OVI Vinculación de una BD a una pagina con PHP.


Información general


Este OVI aborda la temática referente a como conectar una base de datos a una pagina PHP, que hace parte del curso Programación web 1 código 204035, del programa Tecnología en Desarrollo de Software de la universidad Nacional Abierta y a Distancia UNAD.

La temática abordada en este OVI esta diseñada para ser estudiada paso a paso, por lo que recomendamos no saltar ninguno de los temas expuestos en este OVI, y continuar el recorrido en orden de aparición en el menú lateral izquierdo, o haciendo clic en siguiente al final de cada página o sección de este OVI.


Objetivo general


El objetivo general de este OVI es explicar cómo escribir páginas web conectadas a una base de datos, que permita guardar los datos del cliente, o la aplicación, accediendo a ellos a través del lenguaje Backend PHP, el cual es un lenguaje de programación de alto nivel orientado a objetos, con el cual podemos crear aplicaciones o servicios que se despliegan en servidores para la web.


¿Qué vamos a hacer en este OVI?

Es muy sencillo lo que vamos a hacer, son tres pasos donde vamos a adquirir el conocimiento necesario para conectar una base de datos a una pagina PHP y estos son:

Paso 1

Instalar en nuestro equipo de cómputo el SDK de PHP, el servidor de Apache XAMP, que incluye la base de datos MySQL MariaDB, y el intérprete de lenguaje PHP, para correr páginas web escritas en HTML5, JavaScript, y PHP.

Paso 2

Escribir los scripts, clases y páginas web, necesarios para hacer un formulario en una página HTML5, que permita conectarnos a nuestro motor de base de datos y gestionar un CRUD (Creación, lectura, modificación y eliminación), de datos usando un caso de uso de una academia que ofrece cursos informáticos, enfocándonos en la lista de ofertas académicas y sus precios.

Paso 3

Probar el funcionamiento de nuestro código generado, y observar las acciones realizadas desde nuestra página web, y nuestro motor de base de datos.
